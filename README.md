# 🚀 Sensai – AI Career Coach

An AI-powered career development platform that helps users analyze resumes, identify skill gaps, and prepare for interviews with intelligent insights.

---

## ✨ Features

* 📄 **Resume Analyzer**
  Get ATS score, keyword analysis, and improvement suggestions

* 🧠 **Skill Gap Analysis**
  Identify missing skills based on job descriptions

* ✉️ **AI Cover Letter Generator**
  Generate personalized cover letters

* 🎤 **Interview Preparation**
  Practice with AI-generated questions & feedback

* 📊 **Career Dashboard**
  Track your progress and growth

---

## 🛠️ Tech Stack

* **Frontend:** Next.js, React, Tailwind CSS
* **Backend:** Node.js (API Routes)
* **Database:** Prisma
* **AI/ML:** Custom ML services + AI APIs

---

## ⚙️ Installation

```bash
git clone https://github.com/anuradhayadav04/sensai.git
cd sensai
npm install
npm run dev
```

---

## 🔐 Environment Variables

Create a `.env` file:

```env
OPENAI_API_KEY=your_key
DATABASE_URL=your_db_url
```
